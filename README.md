# 🏗️ Компания Строй проект

Современный многостраничный сайт для компании по асфальтированию территорий с backend на FastAPI.

Проект включает:
- Frontend (HTML, CSS, JavaScript)
- Backend (FastAPI + SQLite)
- Обработку заявок с сохранением в базу данных

---

## 🚀 Функционал

### 🌐 Frontend
- Многостраничный сайт:
  - Главная
  - Услуги
  - Работы
  - Контакты
- Современный UI/UX
- Анимации и hover-эффекты
- Адаптивная верстка
- Форма отправки заявки

### ⚙️ Backend
- FastAPI сервер
- REST API
- Сохранение заявок в базу данных (SQLite)
- Валидация данных
- CORS для подключения фронта

### 📩 Заявки
- Отправка с сайта
- Сохранение:
  - Имя
  - Телефон
  - Дата создания

---

## 🛠️ Технологии

### Frontend:
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

### Backend:
- Python 3
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn
